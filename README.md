# Portfolio / CV interactif — Tom Lepert

CV et portfolio data interactif, développé avec Streamlit, mettant en valeur mes projets en Data Analysis, Machine Learning, Automation et Data Engineering.

Ce site présente mes projets sous forme de cartes visuelles, avec des fiches détaillées, des démos, des liens GitHub et une interface stylisée (CSS custom), pour offrir une expérience proche d’un mini-site pro plutôt qu’une simple app Streamlit.

## Objectifs du projet
- Proposer un CV moderne et interactif
- Centraliser mes projets data, ML et engineering

Mettre en valeur :
- le contexte
- la problématique
- la méthodologie
- les technologies
- les résultats

- Offrir une UX claire et agréable pour recruteurs et collaborateurs
- Remplacer un PDF statique par une vitrine dynamique
